# chaa portfolio

My website built with [next.js](https://nextjs.org/) and hosted on [vercel](https://vercel.com)

## 🏗️ Development

1. Installl dependencies

```bash
npm install
```

2. Start the development server

```bash
npm run dev
```

## 🚀 Production

1. Build for production

```bash
npm run build
```

2. Start the production server

```bash
npm run start
```
